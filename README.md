## About

This repository tracks my solutions to two of LeetCode's official learning paths: the LeetCode 75, covering core algorithm and data structure patterns, and the SQL 50, focused on query writing and relational problem solving.

The goal is to practice consistently, document my thinking through comments in each file, and build a reference I can come back to.
