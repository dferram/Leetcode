# LeetCode 75 & SQL50 Solutions

This repository contains my solutions to the problems from the **LeetCode 75** and **SQL50** learning paths on [LeetCode](https://leetcode.com/).

## Structure

- `/javascript/`: Solutions in JavaScript
- `/python/`: Solutions in Python
- `/sql/`: Queries for the SQL50 path

Each file is named after its respective problem number and title from LeetCode.

## Goals

- Complete the LeetCode 75 (Algorithms Learning Path).
- Complete the SQL50 (SQL Learning Path).

## Why?

To practice and document my progress in solving classic programming and SQL problems, improving my skills in both languages.

## How to use this repo

1. Choose the folder for your preferred language (`/javascript/`, `/python/`, or `/sql/`).
2. Open the file for the problem you’re interested in.
3. Each solution usually includes explanatory comments.
